# t1d-sickapp

## How to build the web application
```
// Clone this repository
git clone https://github.com/nickdotreid/t1d-sickapp.git
cd t1d-sickapp

// Install NodeJS and NPM

// On OSX
// Install homebrew frist -- http://brew.sh/
brew install node


// Install Gulp and Bower globally
sudo npm install -g gulp bower

// Install all node packages
npm install

// Build the web application
gulp

// The css files should now be updated
```
